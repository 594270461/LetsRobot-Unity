This is the unity program for zone capture game by using ARToolKit.
* Add ARMarker(Script) to inspector (Hierarchy: ARToolKit).
* Choose Square Barcode as Type.
* Assign Barcode ID compatible with your actual square barcode.
* (Optional) If you want to display something on top of the barcode, you can add ARTrackedObject(Script) to inspector (Hierarchy: Scene root > Marker scene)
